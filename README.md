# Gravity

*3D Physics Engine Written in Go*

## Installation

```
go get -u github.com/go-delve/delve/cmd/dlv
go get -u go.uber.org/zap
```

## Start

```
go run github.com/kwlo/gravity
```

## Run tests

```
CGO_ENABLED=0 go test ./...
```
